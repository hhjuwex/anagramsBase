В данном репозитории содержится базовая версия программы для работы с анаграммами слов и цифровых последовательностей.
Данный файл описывает порядок работы и проверки тематического задания по дисциплине Компьютерная Алгебра.
Автор работы: ст. Жолобов Роман, гр. 5130901/50001

Порядок запуска: нажатие правой кнопкой мыши по файлу main.py -> Открыть с помощью -> Python. После запуска откроется графический интерфейс, предоставляющий функционал работы с программой.
Решение поставляется вместе со словарем nouns_dict1.csv в папке data.

Рядом с файлом README.txt находится файл English.txt. Т.к. функционал программы позволяет загружать свой словарь, данный файл можно использовать в качестве тестирования.
Порядок применения: нажатие правой кнопкой мыши по файлу English.txt -> Копировать как путь.
Скопированный путь можно вставить в соответствующее поле в графическом интерфейсе, для этого нажмите правой кнопкой мыши по полю и используйте сочетание клавиш ctrl + v
Перед этим убедитесь, что у вас стоит английская раскладка клавиатуры.
